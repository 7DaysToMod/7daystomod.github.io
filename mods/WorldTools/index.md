# WorldTools

World Editing commands for Dedicated Servers.

## Dependencies

This mod requires the following mods to already be installed on the server.

[7dtmlib](http://7daystomod.com/mods/7dtmlib)

__wtclosest__

__despawn__

__wtcopy__
